Partial Panel Instrument Flying (CPL)

Objective

To teach:
  • Learn how to control and maneuver the aircraft using partial panel

Motivation

  • If a gyroscopic instrument becomes unsteady or unserviceable, the pilot must be able to continue the flight using partial panel
  • If one of the power sources for the gyroscopic instrument fails, the pilot will have to continue the flight using partial panel
  • Flying VFR and enter cloud – no AI + HI
  • Licensing requirement for CPL license

Threshold Knowledge Test

  • Scan technique: Straight level, Climb, Descent, Turn (remember primary scan?)

Essential background Knowledge

  • System Review:
    1. Static port blockage:
      • Altimeter: Freezes at the current reading
      • VSI: Freezes at zero
      • ASI: Under-read climb / Over-read descent
      • How to solve: use alternate static (C152s do not have alternate static b/c they are not for IFR)
      • How to confirm: climb slightly to check if it is really a static port blockage
    2. Pitot-tube blockage:
      • ASI: (If complete blockage: Under-read descent / Over-read climb) & (If partial blockage: ASI reads zero)
      • How to solve: turn pitot heat on
    3. Vacuum pump failure:
      • AI: Tilts sideways
      • HI: Moves unpredictable
      • How to solve: Check suction gauge(4.5-5.5inHg), verify with TC with AI or MC with HI
    4. Turn Coordinator:
      • Malfunction Flag
      • How to solve: Electric malfunction checklist
  • Rate one turn: 3 degrees per second (Each number on HI is 10 seconds)
    • 180 degree turn = 1min
    • 360 degree turn = 2mins
    • Bank angle = TAS/10 + 7

Procedures

  1. Straight & Level
    • Straight: Primary instrument (TC + MC)
      • Cruise attitude – TC
      • Direction – MC
      • Control yaw & Wings level
    • Level: Primary instrument (TC + Altimeter + VSI)
      • Cruise attitude – TC
      • Altitude – Altimeter & VSI
  2. Climb
    • Entry: Primary instruments (TC + ASI)
      • APT
    • Maintain: Primary instruments (TC + ASI)
    • Approaching desired altitude:
      • Call out 200ft below
      • Scan technique changes, primary instruments become (TC + Altimeter)
    • Recovery: Primary instruments (TC + Altimeter)
      • APT
  3. Descent
    • Entry: Primary instruments (TC + ASI)
      • PAT
    • Maintain: Primary instruments (TC + ASI)
      • Control yaw
    • Approaching desired altitude:
      • Call out 200ft above
      • Scan technique changes, primary instruments become (TC + Altimeter)
    • Recovery: Primary instruments (TC + Altimeter)
      • PAT
  4. Turns
    • Entry: Primary instruments (TC + MC)
      • Set the time (Each number on HI is 10 seconds)
      • Rate 1 turn
    • Maintain: Primary instruments (TC(Time) + Altimeter + ASI)
    • Recovery: Primary instruments (TC(Time) + Altimeter)
      • Roll out at the last second

Safety

  • Small corrections
  • Keep scanning
  • Gentle turn
  • During taxi: confirm if all instruments are working correctly
  • Before turning: ask “clear left/right”
